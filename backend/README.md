# Mentor-Mentee Backend

Setup

1. Copy .env.example to .env and fill values
2. npm install
3. npm run dev

Routes
- /api/auth/register/mentor
- /api/auth/register/mentee
- /api/auth/register/hod
- /api/auth/login
- /api/auth/mentors
- /api/mentors/*
- /api/mentees/*
- /api/tasks/*
- /api/proofs/*
- /api/hod/*
- /api/reports/*
